# NovaLight.Core

NovaLight.Core — это мощный фреймворк для динамической загрузки и управления службами в изолированных контекстах сборок (.NET AssemblyLoadContext) с поддержкой внедрения зависимостей (DI), а также встроенной системой логирования и управления терминалом.

---

## Основные возможности

- **Загрузка Assembly в отдельный контекст**  
  Позволяет загружать сборки в изолированные `AssemblyLoadContext`, что обеспечивает независимость версий зависимостей и возможность динамической подгрузки и выгрузки модулей без влияния на основное приложение.

- **Поддержка DI-зависимостей в загружаемых контекстах**  
  Встроенная интеграция с системой внедрения зависимостей позволяет создавать и управлять службами с их зависимостями внутри каждого контекста, обеспечивая гибкость и модульность.

- **Запуск и остановка служб в изолированных контекстах**  
  Управление жизненным циклом сервисов — запуск, остановка и освобождение ресурсов — реализовано с учётом особенностей изоляции контекстов, что повышает стабильность и безопасность приложения.

- **Система логирования**  
  Встроенный механизм логирования с возможностью централизованного сбора и управления логами из разных контекстов и сервисов.

- **Отдельная сборка NovaLight.Console — красивый терминал с удобным управлением**  
  NovaLight.Console предоставляет современный и удобный терминал для взаимодействия с фреймворком и загруженными сервисами. Он поддерживает удобное управление, отображение логов, интерактивные команды и расширяемость, что делает работу с NovaLight максимально комфортной и эффективной.

---

## Почему NovaLight.Core?

- Использует современные возможности .NET для изоляции и динамической загрузки кода.
- Позволяет работать с разными версиями одних и тех же зависимостей параллельно.
- Обеспечивает удобный и безопасный способ расширения функционала приложения через плагины и модули.
- Интегрируется с DI-контейнерами для удобного управления зависимостями.
- Предоставляет готовые инструменты для логирования и мониторинга.
- Включает удобный терминал NovaLight.Console для управления и мониторинга в реальном времени.
